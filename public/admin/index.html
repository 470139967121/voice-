Maintenance

            

                

                    
System Messages

                    
Delete all conversations with SHYTALK_SYSTEM, including their messages,
                        settings subcollections, and associated images from Storage.

                    
                    

                

                

                    
Reports

                    
Delete all reports (pending, resolved, archived), report locks, and report
                        evidence files from Storage.

                    
                    

                

                

                    
Warnings

                    
Reset warning count, active warning, and Good Character Score to 100 for all
                        users.

                    
                    

                

                

                    
Appeals

                    
Delete all suspension appeals (pending, approved, denied).

                    
                    

                

                

                    
Storage

                    
Audit storage usage and purge orphaned files (PM images, stickers, report
                        evidence). Profile and cover photos are not affected.

                    
                    
                    

                

                

                    
Backpacks

                    
Empty every user's backpack. This removes all gifts from all backpacks
                        permanently.

                    
                    

                

                

                    
Gift Walls

                    
Reset all gift walls. This removes all received gift records from every
                        user.

                    
                    

                

                

                    
Coins

                    
Set shyCoins to 0 for all users.

                    
                    

                

                

                    
Beans

                    
Set shyBeans to 0 for all users.

                    
                    

                

                

                    
Spin History

                    
Delete all gacha spin transaction records and reset pity counters for all users.

                    
                    

                

                

                    
All Transactions

                    
Delete ALL transaction records (gacha, gifts, purchases, etc.) for all users. Processes 30 users per call.

                    
                    

                

                

                    
Super Shy

                    
Remove Super Shy status from all users (sets isSuperShy=false, clears expiry and tier).

                    
                    

                

                

                    
Destroyed Users

                    
Remove user docs corrupted by the batch update bug (missing createdAt). These users can then re-register fresh.

                    
                    

                

                

                    
Device Bindings

                    
Delete all device-to-user bindings. Allows any user to sign in from any device.

                    
                    

                

                

                    
Backfill User Types

                    
Set userType to "MEMBER" for all users that are missing it. Safe to run multiple times.

                    
                    

                

                

                    
Private Messages

                    
Delete all 1-on-1 private message conversations and their associated images from storage.

                    
                    

                

                

                    
Group Chats

                    
Delete all group chat conversations, their messages, and associated media (group photos, images).

                    
                    

                

                

                    
Closed Rooms

                    
Delete all closed rooms and their messages/seat requests. Active and owner-away rooms are preserved. Also runs automatically daily (rooms closed > 7 days).

                    
                    

                

                

                    
Broadcasts

                    
Delete all gift send and gacha win broadcast records.

                    
                    

                

                

                    
Audit Logs

                    
Delete all admin audit log entries.

                    
                    

                

                

                    
Stalkers

                    
Delete all profile visitor (stalker) records from all users and reset stalkerCount and newStalkerCount to 0.

                    
                    

                

                

                    
Suggestions

                    
Delete all suggestions (pending, approved, rejected, merged) and their dispute records.

                    
                    

                

                

                    
Subscriptions

                    
Delete all subscription records and remove active subscription status from all users.

                    
                    

                

                

                    
Notifications

                    
Delete all notification records from all users.

                    
                    

                

                

                    
☢ Identity Graphs

                    
Delete ALL identity graph data including linked identifiers, device fingerprints, and network associations. This will break all identity-based suspensions and cannot be undone.

                    
                    

                

                

                    
⚡ Sync from Production

                    
Wipe ALL dev Firestore data and replace it with a copy of production data.
                        This copies users, rooms, conversations, gifts, economy config, banners,
                        and all subcollections. Cannot be undone.

                        reports, warnings, orphaned storage, backpacks, gift walls, coins, beans, spin history, Super Shy, appeals, private messages, group chats, closed rooms, broadcasts, audit logs, stalkers, suggestions, subscriptions, and notifications.
